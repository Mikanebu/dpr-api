Publish or Download from the command line

Install the data package manager command line tool:

$ [sudo] pip install dpmpy

Configure (use your secret token here):

$ dpmpy configure

Publish your data:

$ cd your-data-package-directory/

$ dpmpy publish

Download:

$ dpmpy download
